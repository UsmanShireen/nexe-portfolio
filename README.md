## 📦 Run Locally

```bash
# Clone karo
git clone https://github.com/UsmanShireen/nexe-portfolio.git
cd nexe-portfolio

# Install karo
npm install

# Run karo
npm run dev
```

Open karo: `http://localhost:3000`
